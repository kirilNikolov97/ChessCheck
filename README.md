# ChessCheck

<b>Info: </b> The program gets the table 8 x 8 from /docs/table.txt and prints if one of the kings is in check/checkmate in the current turn.

-  Lower case letter - White figure
  
-  Upper case letter - Black figure
  
  <b>Type of figures:</b>
  -  p - Pawn
  -  q - Queen
  -  n - Knight
  -  r - Rook
  -  b - Bishop
  -  k - King
